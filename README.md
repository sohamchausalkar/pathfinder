# Pathfinder

## Overview 


Pathfinder AI is an AI-powered tool designed to help students find personalized summer programs, internships, and opportunities based on their interests. The tool uses a decision tree model to match users with relevant opportunities, providing tailored suggestions based on various input factors like field of study, skills, and location preferences.

## Features

1. Personalized recommendations for internships, summer programs, and opportunities. 
2. AI-driven decision tree model for accurate matching. 
3. User input-based filtering for better customization. 
4. Scalable and adaptable to different interest areas.
